# Security Policy

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub security advisory
when the repository is hosted on GitHub. If that is not available, contact the
maintainer directly and avoid posting exploit details in a public issue.

## Scope

This project reads local repository metadata and may execute read-only git
commands. It should not transmit repository contents or credentials.

## Handling Sensitive Data

Do not include tokens, private logs, or confidential repository data in reports
generated for public sharing.
