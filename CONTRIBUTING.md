# Contributing

Thanks for considering a contribution.

## Local Setup

```bash
python -m pip install -e .
python -m unittest
```

## Pull Requests

- Keep changes small and focused.
- Include tests for behavior changes.
- Update README or docs when CLI behavior changes.
- Avoid adding dependencies unless they materially improve the tool.

## Reporting Issues

Please include:

- operating system
- Python version
- command that failed
- expected behavior
- actual behavior

Do not include private repository data, tokens, or credentials in issues.
